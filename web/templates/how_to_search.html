Find your data

The pdbrt.polsl.pl website provides all users option to freely search, explore and view structures with known residence time. The homepage provides some general information about all available structures in tabular format, and have the possibility to download whole sets of data files for result sets consisting of multiple PDBrt entries.


Search Options

Specific information about the molecule(s) of interest can be used to query the 3D structure archive via basic search. Basic search can be run from the homepage by simply typing a search term (PDB ID, protein or ligand name, residence time) into the Search Bar in the middle of the page.

If you wish to view the entire contents of the database, just click on the "go to all complexes" button below the Search Bar.




Query Results

Once a search query has been executed using Bar Search "go to all complexes" clicked, you will be taken down the page to the table with the results list. The results show information such as PDB ID, protein name, ligand name and residence time in minutes. Once a structure of interest has been identified you can examine it carefully and/or download structure related files for exploration and analysis. Just click on related PDB ID. Each structure in the Search Results table opens a Structure Detail page which provides detailed information about a binding kinetics.
